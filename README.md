# Auto File Renamer Pro 🚀

A professional tool for print shops to batch rename files with party codes, dimensions, and quantities.

## 🎯 Features
- ✅ Auto-detect party folders (case-insensitive)
- ✅ Rename: `26_Banner (C.S)(FT.3x6)(Q.2)%8%[ok].jpg`
- ✅ Finalize Mode: Edit `(FT.)`, `(Q.x)`, `%cat%`
- ✅ Subfolder support
- ✅ Auto-update check
- ✅ Works with SQL billing software

## 📦 Download
[Download Latest .exe](https://github.com/jigsthummar007/AutoFileRenamer-Pro/releases/latest)

## 🔧 How to Use
1. Click `📁 Select 2025 Folder`
2. Files appear in list
3. Click `✅ Rename Selected` or `📁 Batch Rename`
4. File moves to `Done` folder

## 🔄 Auto-Update
The app checks for updates on startup. To publish a new version:
1. Update `version.txt` to new version
2. Upload new `.exe` in [Releases](https://github.com/jigsthummar007/AutoFileRenamer-Pro/releases)
